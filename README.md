 🍴 FlavourFolder — Your Digital Recipe Book

 Theme: Your Interactive Kitchen Companion
FlavourFolder is a vibrant, interactive culinary hub designed to bridge the gap between messy handwritten notes and the convenience of a modern web application. It is a centralized kitchen companion where food lovers can organize dishes and discover new flavors.

 Key Features
Smart Recipe Search: A dynamic search bar that filters through recipe cards in real-time as you type.
Dynamic Kitchen Notes: Interactive updates that allow you to toggle between ingredients and personal cooking secrets without a page refresh.
"Chef-Proof" Submitter: A submission form with custom JavaScript validation to ensure all recipe entries are accurate.
Responsive Layout: Built with a professional Bootstrap layout to ensure a perfect experience on any screen size.

 Usage Instructions
1. Open `index.html` in your web browser to launch the application.
2. Use the search bar in the Hero section to instantly find recipes.
3. Navigate to the "Submit" page to add your own culinary masterpieces.
